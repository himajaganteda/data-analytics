# Retail Sales Analysis — Excel Dashboard

## Overview
Analysis of 1,000 retail orders across 15 countries to identify 
business performance patterns using Excel pivot tables, charts, 
and an interactive dashboard.

## Business Questions Answered
1. Which countries underperform in sales?
2. Which product categories are most profitable?
3. Which sales manager generates the highest revenue?
4. What is the relationship between device type and order value?

## Key Findings
- **Portugal** generated the highest revenue (€27.8M); **Italy** the lowest (€935K)
- **Clothing** is the most profitable category (€2.9M total profit)
- **Celine Tumasian** is the top-performing sales manager (€27.8M revenue)
- **PC** users have the highest average order value (€114,200)
